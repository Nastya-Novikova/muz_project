README
Для запуска приложения, необходимо запустить скрипт run.ps1. После чего приложение будет доступно по адресу localhost:80.
Для выполнения скрипта необходимо иметь запущенный Docker Desktop. 
В рамках скрипта будет производиться пересборка приложения, поэтому необходимо иметь:
1. установленный пакет инструментов dotnet,
2. npm, который в случае отсутствия на устройстве, будет установлен.
